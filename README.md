## Project Overview - 
  This project presents an end-to-end Healthcare Analytics solution built using Power BI. The dashboard provides actionable insights into patient demographics, 
hospital performance, lab test trends, and medication usage. It is designed to help healthcare stakeholders make data-driven decisions for improving operational 
efficiency and patient care.

## Business Objective -

### To analyze clinical data and deliver insights on:

- Patient distribution and disease trends
- Hospital revenue and visit patterns
- Department-wise performance
- Lab test utilization
- Medication usage and treatment duration

## Key Insights - 

- Identified top revenue-generating departments
- Analyzed patient distribution across conditions
- Observed trends in lab test usage over time
- Evaluated medication patterns and treatment durations

## Tools & Technologies -

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling (Star Schema)
